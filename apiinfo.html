the konvrt api

konvrt is equipped with a simple yet powerful API for developers. With the konvrt API, developers can harness konvrt's algorithms in their own applicatons. Using the API is very simple, just generate an API call URL below!

To generate an API call, follow these steps:

Step 1: Enter the quantity.
Step 2: Select the unit category from the drop down menu.
Step 3: Select the unit which you want to convert from.
Step 4: Select the unit which you want to convert to.
Step 5: Click on the Generate URL button.

A pop-up dialog will appear with the API call URL. This URL will return the result (just the quantity) in plaintext format.
